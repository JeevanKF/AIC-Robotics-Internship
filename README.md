# Project Overview

This repository contains my work completed during the AI for Industry Challenge (AIC) robotics internship.

The project involved setting up the official AIC robotics simulation environment, understanding the ROS 2 based architecture, exploring robot control policies, and implementing a custom policy named JeePolicy.

# What I Learned

This project helped me gain practical exposure to robotics software development and simulation workflows.

### Project Highlights

Set up the AIC robotics simulation environment
Worked with ROS 2 and Gazebo-based robotics workflows
Explored existing robot control policies
Created a custom policy named JeePolicy
Modified robot arm motion behavior and execution timing
Used Git and GitHub for version control and project management
